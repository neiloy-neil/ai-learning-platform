# AI Learning Platform

Modern frontend starter built with Next.js App Router, TypeScript, Tailwind CSS, reusable UI primitives, and dark mode support.

## Structure

- `app`: route entrypoints and global styling
- `components/ui`: reusable presentational primitives
- `components/layout`: app shell and shared layout pieces
- `features`: feature-oriented modules
- `lib`: shared utilities and types

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run format`
