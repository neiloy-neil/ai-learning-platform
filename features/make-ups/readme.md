
# Make-Ups

This feature is responsible for managing make-up lessons for absent students.

## Purpose

The main purpose of this feature is to provide a structured workflow for handling student absences and ensuring that they have the opportunity to catch up on missed work.

## Main Components

*   **Absence Trigger:** An absence recorded by the teacher triggers the make-up workflow.
*   **Missed Lesson Record:** A record of the missed lesson, including the concepts that were covered.
*   **Make-Up Options:** A set of options for the parent to choose from, including attending another class, collecting a booklet, or completing an online catch-up lesson.
*   **Completion Tracking:** A way to track the completion of the make-up lesson or catch-up work.
