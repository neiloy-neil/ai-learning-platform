
# Admin Dashboard

This feature is responsible for providing admins with a single operational control centre.

## Purpose

The main purpose of this feature is to give admins a high-level overview of the entire operation and to provide them with the tools they need to manage the centre efficiently.

## Main Components

*   **Today's Overview:** A summary of the classes, attendance, make-up lessons, and diagnostics for the current day.
*   **Pending Actions:** A list of urgent tasks that require the admin's attention, such as new leads, unpaid invoices, and enrolment approvals.
*   **Key Metrics:** A set of key metrics to track the performance of the centre.
*   **Quick Actions:** A set of shortcuts to the most common admin tasks.
