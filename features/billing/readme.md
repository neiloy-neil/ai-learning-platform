
# Billing and Payments

This feature is responsible for managing invoices, payments, and other billing-related tasks.

## Purpose

The main purpose of this feature is to reduce the manual administration of billing and payments and to provide a clear and transparent billing experience for parents.

## Main Components

*   **Fee Rules:** A set of configurable rules for calculating fees, including pro-rata fees, discounts, and referral credits.
*   **Invoicing:** A system for generating and sending invoices to parents.
*   **Payment Tracking:** A system for tracking payments and updating the status of invoices.
*   **Overdue Dashboard:** A dashboard to track overdue invoices and send reminders.
