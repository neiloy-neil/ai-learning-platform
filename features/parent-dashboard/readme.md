
# Parent Dashboard

This feature is responsible for providing parents with a clear view of their children's progress.

## Purpose

The main purpose of this feature is to keep parents informed and engaged in their children's learning journey.

## Main Components

*   **Child Progress Snapshot:** A summary of the child's current class, level, recent assessment results, attendance, and homework status.
*   **Invoices & Payments:** A view of outstanding invoices, payment history, and make-up options for missed classes.
*   **Communication:** A place to view messages from teachers and admins, and to request class changes and make-up lessons.
