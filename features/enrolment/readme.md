
# Classes and Enrolment

This feature is responsible for managing classes and student enrolments.

## Purpose

The main purpose of this feature is to allow admins to manage classes and enrol students in them.

## Main Components

*   **Class Management:** A UI for creating, editing, and deleting classes.
*   **Enrolment Management:** A UI for enrolling students in classes, managing waitlists, and handling class changes.
