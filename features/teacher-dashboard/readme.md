
# Teacher Dashboard

This feature is responsible for providing teachers with the tools they need to manage their classes and students.

## Purpose

The main purpose of this feature is to give teachers a single place to view their classes, take attendance, assign homework, record observations, and view student progress.

## Main Components

*   **Class View:** A view of the teacher's classes, with rosters and links to other actions.
*   **Attendance:** A modal to take attendance for a class.
*   **Homework:** A way to assign and track homework.
*   **Observations:** A form to record observations about students.
*   **Class Readiness Report:** A report that shows the teacher how prepared their class is for the next lesson.
*   **Teaching Resources:** A collection of lesson plans, booklet references, and other materials.
